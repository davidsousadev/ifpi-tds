Seleciona a pasta pelo cmd
comando para iniciar o git na pasta

>>> git init
 
para verificar o status dos arquivos 

>>> git status

(se tiver vermelho, adiciona o arquivo e faz o commit)
adicionar o arquivo no controle de versão

>>> git add $%$#%$#%$#%(nome do arquvo)

adicionar um commit

>>> git commit -m "$#%%%$#%#%$%"

enviar as alterações

>>> git push -u origin master

se nao tiver adicionado adiciona as credenciais

>>>git config --global user email "$%$%"
>>>git config --global user name "ddggfg"
